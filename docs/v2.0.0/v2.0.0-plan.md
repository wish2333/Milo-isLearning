# V2.0.0 实现计划 — 个人记忆闭环(修订版 V2.1)

> **For Claude:** REQUIRED SUB-SKILL: Use `superpowers:executing-plans` to implement this plan task-by-task.
> 对应设计文档:[`v2.0.0-design.md`](./v2.0.0-design.md)(**V2.1**,第二轮 review 定向修正)
> ⚠ 本 plan 的 task 细节以 design V2.1 为准。如 task 描述与 design V2.1 冲突,以 design 为准。

**Goal:** 在 localhost 自托管的 production 模式下,交付"个人记忆闭环"(FSRS 派生缓存 + 今日复习 + 学习反馈),并补齐部署可靠性 + AI 扩充导入 + 客户端搜索。

**Architecture:** 4 Phase 顺序交付。P0(localhost 可靠性)复用现有 createSnapshot;P1(FSRS)用协调层 rebuildScheduleForSlot + 始终维护 schedule;P2(X1)走独立 pipeline + anchor 按 ID 匹配;P3(客户端搜索)rebuild-on-open。

**Tech Stack:** Next.js 15 + React 19 + Zustand + SQLite(localhost)+ ts-fsrs@5.4.1(FSRS-6)

**总周期:** ~5-6 周

**⚠ 执行注意:** Plan 中的文件路径基于设计时已知信息。执行每个 task 前必须用 codegraph/grep 核实真实路径(尤其是 page 组件,逻辑可能在 page.tsx 而非独立组件)。
