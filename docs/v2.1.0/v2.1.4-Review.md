## 2. 决策记录

### 2.1 费曼步骤复用 Quiz 编辑器

费曼步骤与 Choice/Fill Blank 题结构足够接近，采用包装成 Quiz view model 的方式复用既有 `AnswerCorrector`，避免出现两套答案编辑语义。保存时仅将支持的字段映射回 `FeynmanStep`。

### 2.2 作答历史记录用户文本

`FeynmanAttempt.stepResults` 增加可选 `userAnswer`，旧数据仍可读取；旧记录没有文本时显示兼容提示，不阻塞历史浏览。

### 2.3 选择题正确答案始终可见

提交反馈阶段明确展示正确答案，避免用户只看到自己选择的错叉而无法定位正解。对未传 `submittedAnswer` 的旧入口，组件使用内部 selected 值兜底。

### 2.4 第 6 步保持独立语义

最终开放输出仍由 Feynman Eval + Rubric 负责；本版本只将前 5 步纳入题目纠正，不把开放文本误当作选择题答案编辑。
