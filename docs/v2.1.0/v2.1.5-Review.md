## 2. 关键决策

### 2.1 会话编排与模块进度分层

主题会话仍负责编排当前模块和用户主动跳过/重新进入；模块级进度只用于展示兜底，不反向改写会话，避免出现新的双向同步副作用。

### 2.2 重刷页保留 GlobalNav

`src/app/learn/layout.tsx` 已统一提供 `GlobalNav`。主题重刷页不再包裹旧 `LearnShell`，与单库重刷页保持同一页面壳层，消除双层导航。

### 2.3 固定 action bar

提交按钮由各题型组件渲染，反馈后的下一步按钮由流程页渲染；两者统一使用 `QuizActionBar`，避免重构题型内部状态机。固定层只在当前动作存在时挂载，页面本体预留底部空间。
